# Portal AUTP API Documentation

REST API untuk sistem manajemen klaim asuransi padi Jasindo (Portal AUTP).

## Setup & Installation

### Requirements
- PHP 8.3+
- PostgreSQL 14+
- Composer
- Docker (optional)

### Local Setup

```bash
# Install dependencies
composer install

# Generate JWT secret
php artisan jwt:secret

# Setup database
php artisan migrate:fresh --seed

# Start dev server
php artisan serve --host=localhost --port=8000
```

Database akan di-seed dengan:
- 1 admin user: `username: admin`, `password: admin123`
- 30 sample claims dengan berbagai status

### Docker Setup

```bash
# Build and run
docker-compose up -d

# Run migrations and seeders inside container
docker-compose exec app php artisan migrate:fresh --seed
```

Server accessible at `http://localhost:8000`

## Error Handling

API returns standard HTTP status codes:

- **200** - OK
- **201** - Created
- **204** - No Content
- **400** - Bad Request
- **401** - Unauthorized
- **404** - Not Found
- **422** - Unprocessable Entity (validation error)
- **500** - Internal Server Error

All errors return JSON response:
```json
{
  "message": "Error description"
}
```

## Architecture

### Directory Structure
```
app/
  Http/
    Controllers/API/
      AuthController.php
      ClaimController.php
    Requests/
      LoginRequest.php
      CreateClaimRequest.php
      UpdateClaimRequest.php
    Resources/
      ClaimResource.php (API response transformation)
    Middleware/
      JWTAuthenticate.php
  Models/
    User.php
    Claim.php
  Services/
    AuthService.php
    ClaimService.php
  
database/
  migrations/
  seeders/
    UserSeeder.php
    ClaimSeeder.php

routes/
  api.php        (API routes)
  web.php        (Web routes)
```

### Stack
- **Framework**: Laravel 13
- **Authentication**: JWT (tymon/jwt-auth)
- **Database**: PostgreSQL + Eloquent ORM
- **API Documentation**: l5-swagger (OpenAPI/Swagger)
- **Docker**: PHP-FPM + Nginx + PostgreSQL
