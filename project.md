1.	Buatlah aplikasi web sederhana menggunakan React/Vue JS sebagai front end dan Laravel sebagai BackEnd yang dapat melakukan operasi CRUD (Create, Read, Update, Delete) 
2.	Aplikasi diharapkan dapat terhubung ke database SQL (misalnya, MySQL atau PostgreSQL)
3.	Buat struktur dasar aplikasi React dengan beberapa komponen, termasuk satu komponen induk dan beberapa komponen anak. Pastikan untuk mengirimkan prop dari komponen induk ke komponen anak.

5.	Dapat menerapkan lapisan keamanan pada aplikasi yang dibangun

9.	Buat komponen "Responsive Layout" yang menyesuaikan tata letaknya berdasarkan lebar layar. Pastikan desainnya responsif dan berfungsi dengan baik pada perangkat seluler/mobile.
10.	Buat komponen React yang menggunakan fetch atau axios untuk mengambil daftar pengguna dari API endpoint tertentu. Tampilkan data pengguna dalam daftar di dalam komponen.
11.	Buat fitur yang memungkinkan pengguna mencari data berdasarkan nama atau ID. Implementasikan fungsi pencarian dan perbarui tampilan data sesuai dengan hasil pencarian.
12.	Implementasikan pengaturan halaman dan tampilkan data pengguna dalam beberapa halaman dengan sistem pagination
13.	Dokumentasikan APInya menggunakan Swagger/postman
14.	Aplikasi akan di demokan sesuai dengan jadwal yang akan kami sampaikan kemudian.
